# Highcharts Grid - Next.js Example

This is a Next.js example demonstrating the integration of Highcharts Grid Lite and Grid Pro with React.

## Getting Started

First, install dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

This example demonstrates:
- Grid Lite integration with Next.js
- Grid Pro integration with Next.js
- Client-side rendering with Next.js App Router
- TypeScript support

## Build

To build the production version:

```bash
pnpm build
```

To start the production server:

```bash
pnpm start
```
